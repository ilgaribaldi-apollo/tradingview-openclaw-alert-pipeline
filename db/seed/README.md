# Seed Data

Keep seed files safe, explicit, and auditable.

This folder is for:
- example watchlists
- reference promotion states
- local development bootstrap rows
- temporary runtime bootstrap rows while the promotion lane is not yet writing Neon directly

Included examples:
- `watchlist.example.sql` — current default Coinbase watchlist rows
- `runtime_strategy.example.sql` — safe development bootstrap for `strategy-rsi@local-v1`

Do not store secrets here.
